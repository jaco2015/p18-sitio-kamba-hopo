### Favicon SVG spec

- Viewbox: `0 0 32 32`
- Background: círculo `fill="#040A07"`
- Texto: "KH" en `fill="#4A7C59"`, font-family serif, centrado
- Tamaño mínimo legible: 16x16

### Nota sobre favicon.ico (AC: 2)

El SVG favicon cubre browsers modernos. Para legacy fallback, el browser buscará `/favicon.ico` automáticamente. Se puede generar un `.ico` desde el SVG pero no es crítico — browsers modernos prefieren SVG.
