### Vercel Analytics (recomendado)

```html
<!-- En <head>, con defer -->
<script defer src="/_vercel/insights/script.js"></script>
```

Vercel Analytics se activa automáticamente cuando el proyecto está en Vercel. Solo requiere el script tag. Es privacy-friendly (no usa cookies, no recolecta PII).

**Alternativa:** Si se prefiere GA4, se necesita `gtag.js` con ID de medición.

### JS módulo 9 — WhatsApp Click Tracking (Architecture §5.1)

```javascript
/* ═══ ANALYTICS ═══ */
(() => {
  const waLinks = document.querySelectorAll('a[href*="wa.me"]');
  if (!waLinks.length) return;
  waLinks.forEach(link => {
    link.addEventListener('click', () => {
      // Vercel Analytics custom event
      if (window.va) window.va('event', { name: 'whatsapp_click' });
    });
  });
})();
```

Insertar como último módulo JS (después de Hamburger Menu, módulo 8).
